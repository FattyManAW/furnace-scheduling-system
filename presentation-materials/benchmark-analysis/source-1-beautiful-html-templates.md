## 🔧 Design Tokens Extract

### Typography Patterns
- **Serif Headlines**: Cormorant Garamond, Playfair, Source Serif 4, Instrument Serif, Lora, Bodoni
- **Sans Body**: Jost, Inter, Archivo, Syne, Bricolage
- **Display**: Shrikhand, Bebas Neue, Alfa Slab, Fraunces
- **Handwritten**: Caveat, Caveat Brush

### Color Palette Patterns
- **Naval/Corporate**: Deep navy + gold/teal accent (Signal, Vellum, Emerald)
- **Warm/Paper**: Cream/ivory base + single saturated accent (Blue Professional, Bold Poster)
- **Pastel Pop**: Multi-color on neutral base (Capsule, Creative Mode, BlockFrame)
- **Editorial Dark**: Black/dark canvas + hot accent (Pink Script, Studio, Broadside)

### Layout Patterns
- **Split Layout**: 60/40 or 50/50 text/image
- **Full Bleed**: Edge-to-edge color block + centered headline
- **Card Grid**: 2×2 or 3×2 modular cards
- **Timeline**: Horizontal/vertical milestone track
- **Statement**: Single large quote or number per slide
