## 提取的設計模式（跨 7 大產業）

### 1. 通用 Landing Page 佈局
```
Hero（全屏） → Features（3-4 cards） → Stats（數字） → CTA → Footer
```
適用：P0 用友商務簡報的「解決方案」slide

### 2. 色彩系統（產業視角）
| 產業 | 主色 | 輔色 | accent |

| 金融 | #1E3A5F (海軍藍) | #F8F9FA (白) | #D4AF37 (金) |

### 3. 響應式斷點（nicepage 通用）
```
Mobile:  < 768px  → 單欄 · 堆疊式
Tablet:  768-1024 → 雙欄 · 側欄
Desktop: > 1024px → 多欄 · 大留白
```

## 對簡報設計的啟發

- nicepage 模板是 **網站**（非簡報），但 design tokens 可直接用於我們的簡報
- Hero section → 用友簡報封面
- Stats section → KPI slide
- Feature cards → 產品特性 slide
- 產業色彩系統 → 不同客戶群的主題配色
