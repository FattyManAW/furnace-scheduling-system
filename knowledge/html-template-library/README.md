## 使用原則

1. **選擇模板** → index.json 按風格/色系/用途匹配
2. **複製 HTML** → 從 `templates/` 選取對應範本
3. **替換內容** → 保留 CSS 結構，只換文字與圖片
4. **引用 shared/** → 用我們的 design-tokens.css 覆蓋預設配色

## 對我們的簡報系統的適用性

- **P0 用友簡報**：推薦 `editorial-forest`（商務 · 深綠）或 `emerald-editorial`（雜誌風格）
- **P3 CRIS 簡報**：推薦 `cobalt-grid`（科技感 · 藍調）或 `neo-grid-bold`（neo-brutalism）
- **P3 APS 簡報**：推薦 `velvet-night`（深色優雅）或 `editorial-tri-tone`（三色簡約）
- **Kanban/OTD Dashboard**：推薦 `soft-editorial`（柔和數據）或 `stencil-tablet`（資訊密度高）
