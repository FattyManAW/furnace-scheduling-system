## 架構

- `metadata.json` — 模板元數據（風格/色系/用途）
- `screenshots/` — 3 張截圖（cover · mid · later）

## 設計系統共通模式

### 色彩策略
- **三色規則**：主色 60% + 輔色 30% + accent 10%
- **CSS 變數系統**：`--c-primary` · `--c-bg` · `--c-text` · `--c-accent`
- **無框線設計**：用留白和色彩分割，不用 border

### 字體策略
- **Display 字體**（標題）：Cormorant Garamond · Playfair Display · Bodoni
- **Body 字體**（內文）：Source Serif 4 · Inter · Lato · Noto Sans
- **Mono 字體**（數據）：DM Mono · JetBrains Mono
- **配對規則**：Serif heading + Sans body（最常見）或 Sans heading + Serif body

### 佈局策略
- **Cover slide**：大標題居中 + accent 線條 + 日期/作者資訊
- **Content slide**：2 欄 (50/50) 或左文右圖 (40/60)
- **Data slide**：KPI 數字大字 + 簡短說明
- **Closing slide**：感謝 + CTA + 聯絡資訊

### 動畫策略
- **fade-up**：逐項淡入（>90% 模板採用）
- **scale-in**：數字從 0 滾動
- **stagger**：grid 子項依序進場
- **無自動播放**：使用者控制節奏

### CSS 變數提取（通用模板）

```css
:root {
  /* 來源：beautiful-html-templates 34 套模板共通模式 */
  --c-primary: #1a3a2a;      /* 主色（可替換） */
  --c-secondary: #d4a5a5;    /* 輔色 */
  --c-accent: #f0c040;       /* accent 色 */
  --c-bg: #faf8f5;           /* 背景（暖紙色） */
  --c-text: #2c2c2c;         /* 主文字 */
  --c-text-muted: #6b6b6b;   /* 次要文字 */
  --c-border: #e0ddd5;       /* 極細邊框 */
  --f-display: 'Cormorant Garamond', serif;
  --f-body: 'Source Serif 4', serif;
  --f-mono: 'JetBrains Mono', monospace;
  --s-base: 1rem;
  --s-gap: 2rem;
  --s-section: 4rem;
}
```
