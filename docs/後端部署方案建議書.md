## 2. 方案評估

### 2.1 方案對比

| 方案 | 月費 | 部署難度 | 自動擴展 | 靜態 IP | 推薦度 |
|------|------|----------|----------|---------|--------|
| **Docker Compose（自管 VPS）** | ~$6/mo | 🟡 中 | ❌ | ✅ | ★★★★★ |
| Railway | $5/mo 起 | 🟢 低 | ✅ | ✅ | ★★★★ |
| Render | 免費層 | 🟢 低 | ❌（sleep 15min） | ✅ | ★★★★ |
| Fly.io | $3/mo | 🟡 中 | ✅ | ✅ | ★★★★ |
| AWS EC2 t4g.nano | ~$5/mo | 🔴 高 | ❌ | ✅ | ★★★ |
| Vercel Serverless | $20/mo | 🔴 高 | ✅ | ❌ | ★★ |

### 2.2 推薦方案：Docker Compose + VPS

**理由**：
- SQLite 是嵌入式資料庫，最適合單機部署（不需要外部 DB 服務）
- 資料量小（113KB JSON + < 10MB SQLite），最低規格 VPS 即可
- Docker Compose 一鍵啟動，維運簡單
- 成本最低（Hetzner VPS $4/mo 或 DigitalOcean $6/mo）
- 可同時部署 Dockerfile，隨時切換到 Railway/Render/Fly.io

## 附錄 A：方案詳細對比

### Railway
- ✅ GitHub 連動自動部署
- ✅ 內建 HTTPS + 自定義域
- ✅ 健康檢查
- ❌ SQLite 限制（ephemeral disk，重啟後資料可能消失 — 需掛 volume）
- 💰 $5/mo 起

### Render
- ✅ 免費層（每月 750h）
- ✅ GitHub 連動自動部署
- ❌ 15 分鐘閒置後休眠（冷啟動 30-60s）
- ❌ SQLite 同 Railway 問題（需 disk mount）
- 💰 免費 / $7/mo 起

### Fly.io
- ✅ 全球 edge 部署
- ✅ 內建 volume 支援 SQLite 持久化
- ❌ CLI 部署流程較複雜
- 💰 $3/mo 起

### VPS + Docker Compose
- ✅ 完全控制
- ✅ SQLite 無任何限制
- ✅ 可掛 nginx + certbot 做 HTTPS
- ❌ 需手動設定 HTTPS / 防火牆 / 更新
- 💰 $4-6/mo
